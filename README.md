# RepoExplorer

RepoExplorer is the ultimate tool for discovering high-quality, running GitHub repositories, designed to help developers find source code that is up-to-date, reliable, and ready to use. With its advanced search and filtering options, RepoExplorer ensures you can quickly find the best repositories tailored to your needs.

## Features
- **AI-Powered Code Source Generator:** Utilize AI to find the most relevant, well-maintained, and smooth-running repositories.
- **Advanced Search:** Type natural language prompts to discover projects based on specific criteria such as licenses, stars, programming languages, and more.
- **Filter Options:** Refine your search by selecting a programming language or sorting repositories by stars, forks, or recent updates.
- **Pagination:** Easily browse through large sets of results.
- **Skeleton Loading:** Enjoy visual feedback while results are fetched, ensuring a seamless experience.
- **Intuitive UI:** Navigate with ease through a clean, responsive interface featuring sidebars for quick access.

## Example Prompts
Here are some example prompts you can use with RepoExplorer to discover the best repositories:

- **Find blockchain projects with the MIT license on GitHub.**
- **Show React projects with over 100 stars on GitHub.**
- **List Python repositories for machine learning, sorted by stars.**
- **Give me some blockchain projects with MIT license on GitHub.**
- **Find top blockchain projects with the MIT license on GitHub.**

## How It Works
1. **Search**: Enter your query in natural language into the search bar.
2. **Filters**: Apply filters based on programming language, license type, stars, or date of the last update to ensure you're getting the best, most relevant code.
3. **Results**: Browse the top repositories, view critical details like stars, forks, open issues, and licenses, and visit the repository directly from the results page.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/repoexplorer.git
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   flask run
   ```

## Technologies Used
- **Flask** for the backend.
- **HTML/CSS/JavaScript** for the frontend.
- **AI Code Source Generator** for ensuring that the best running, non-outdated repositories are found.
